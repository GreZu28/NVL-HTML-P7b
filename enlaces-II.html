A

address
Elemento de bloque, elemento contenedor. El elemento de dirección se utiliza ocasionalmente; permite a los visitantes de un sitio Web ponerse en contacto con el autor de la página. Esta información puede ser una dirección de correo electrónico, un enlace Web (las opciones más habituales)o bien una dirección postal
area
Solo permitido en map, elemento independiente. El elemento "area" define una región en la que se puede hacer clic (llamada punto caliente) dentro de un mapa de imagen (que se genera con el elemento map). Cuando se define un área, debe aportar la URL de destino (con el atributo href), el tipo de forma (con shape) y las coordenadas de esa forma (con coords). Para shape puede especificar un círculo, un cuadrado o un polígono. Para el primero de ellos, las coordenadas se especifican en el orden siguiente: punto central (coordenada x), punto central (coordenada y), radio. Para cualquier otra forma, se proporcionan las esquinas en orden, como una serie de coordenadas x-y: x1,y1,x2,y2, etc

B

b
Elemento de línea, elemento contenedor. Este elemento muestra texto en negrita. La norma oficial sugiere el uso de b en un texto desplazado estilísticamente, el cual debe presentarse en negrita pero que no tiene mayor importancia que el resto del contenido, como pueden ser las palabras clave, los nombres de productos o cualquier otro elemento que imprimiría en negrita.
button
El elemento button permite originar un botón sobre el que se puede hacer clic dentro de un formulario, con el contenido que desee dentro de él (por ejemplo, puede colocar una frase o una imagen entre sus etiquetas de inicio y fin). Al igual que con cualquier otro control de formulario, debe proporcionar un nombre único y un valor que el formulario enviará cuando un visitante haga clic en el botón.

C

caption
Solo permitido en table, elemento contenedor. El elemento caption define el texto del título de una tabla. Si lo utiliza, deberá colocarlo como primero dentro de un elemento table.

D

dd
Descripción de diccionario. Solo permitido en dl, elemento contenedor. El elemento dd dentifica la descripción en una lista de diccionario.
dfn
El elemento apenas se utiliza e indica la instancia que sirve para definir un término. Por ejemplo, la primera vez que ve una palabra nueva en un libro, éste aparece en cursiva. Se debe a que es la instancia que hay que determinar y suele ir seguida de una definición. El navegador muestra el elemento en cursiva.

E

em
Puede tener dos significados:
  • Es una unidad de medición con escalabilidad utilizada en diseño web, relativa al contenedor superior inmediato. Por ejemplo, si hay un div que tiene un tamaño de fuente de 17 px, los elementos contenidos dentro de este tendrán una fuente con cuerpo de 1 em, a menos que se especifique algo diferente en el CSS.
  • Una etiqueta HTML para definir que a un texto se le debe dar cierto énfasis semánticamente hablando. De forma predeterminada tiene un estilo CSS itálico, el cual podemos modificar en la hoja de estilos.

F

figure
El elemento figure envuelve una imagen y su título asociado.
form
Representa un formulario, consistendo de controles que puede ser enviado a un servidor para procesamiento.

G

H

head
Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo.
html
El elemento html es el primero de cualquier documento HTML. Envuelve el resto del conjunto. Si origina una página Web normal, el elemento html contiene dos ingredientes esenciales más: el elemento head, que define el título, los metadatos y las hojas de estilo vinculadas, y el elemento body, que alberga el contenido real.

I

iframe
El elemento iframe genera un marco en línea, una ventana incrustada y desplazable que muestra una página Web dentro de otra. Se proporcionan los atributos src (la página que se desea mostrar en el marco), id (el nombre unívoco del marco) y width (ancho) y height (alto) (las dimensiones del marco en píxeles). También puede desactivar el borde estableciendo el atributo frameborder="0" o desactivar el desplazamiento añadiendo el atributo scrolling="no".

J

K

kbd
Representa la entrada de usuario, por lo general desde un teclado, pero no necesariamente, este puede representar otras formas de entrada de usuario, como comandos de voz transcritos.

L

link
El elemento link describe una relación entre el documento actual y otro. Por ejemplo, puede utilizarlo para apuntar a la versión anterior del archivo actual. Más comúnmente, se emplea para señalar a una hoja de estilo externa que ofrece instrucciones de formato para la página actual. El elemento link siempre se coloca en la sección head de una página.

M

main
Define el contenido principal o importante en el documento. Solamente existe un elemento main en el documento.
meta
Los metaelementos permiten incrustar información descriptiva en las páginas Web. Los visitantes nunca ven esta información, pero los programas automatizados como los motores de búsqueda pueden encontrarla cuando escaneen su sitio Web. Añadimos metadatos colocando elementos meta en la sección head de la página. Cada elemento meta incluye un atributo id (que identifica el tipo de información que se añade) y un atributo content(que proporciona la información en sì). Aunque puede tener un número ilimitado de potenciales elementos meta, los dos màs comunes son la descripción y las palabras clave

N

nav
Etiqueta HTML que define una serie de links de navegación dentro de una página. Entre los usos que se le puede dar, está el menú superior de navegación, una tabla de contenidos (como las utilizadas en artículos de Wikipedia), la paginación, entre otros. El elemento nav representa una sección de la página que contiene enlaces, los cuales pueden apuntar a temas en la página actual o de otras páginas de un sitio Web. De hecho, es habitual que una página incluya varias secciones nav: una para el menú de navegación del sitio, otra para temas del artículo actual, etc. Conviene destacar que no es necesario envolver todos los bloques de enlaces en elementos nav. En su lugar, los diseñadores Web suelen reservarlo para las secciones de navegación más importantesde una página. Puede ubicar nav dentro de otro elemento HTML5.

O

object
Representa un recurso externo, que será tratado como una imagen, un sub-documento HTML o un recurso externo a ser procesado por un plugin.
ol
Una lista ordenada comienza con el elemento ol y contiene múltiples elementos de lista, cada uno de ellos representado por el elemento li. En una lista ordenada, el navegador numera cada uno consecutivamente, empleando su selección de números, letras o números romanos

P

p
El elemento de párrafo alberga un párrafo de texto
pre
Indica que su contenido esta preformateado y que este formato debe ser preservado.

Q

q
Define una cita corta dentro de otro elemento de bloque, como un párrafo

R

S

svg
Define una imagen vectorial embebida.

T

table
El elemento table es el elemento más externo que define una tabla. Dentro de él se definen filas mediante tr y en el interior de cada fila, se recurre al elemento td para definir las celdas individuales y especificar su contenido.
tbody
Representa el bloque de filas que describen los datos contretos de una tabla.
thead
Representa el bloque de filas que describen las etiquetas de columna de una tabla.
tfoot
Representa los bloques de filas que describen los resúmenes de columna de una tabla.

wbr
Representa una oportunidad de salto de línea, es decir, un punto sugerido de envoltura donde el texto de múltiples líneas puede ser dividido para mejorar su legibilidad.
